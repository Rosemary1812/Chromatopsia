# DevCanvas — 功能愿景

> 本文档描述 DevCanvas 需要实现的功能和特性，不涉及技术架构和实现细节。
> 架构、目录结构、技术选型另见 `architecture.md`（待编写）。

---

## 1. 画布与项目管理

### 1.1 无限画布
- 无限延展的 2D 画布，支持自由摆放 UI 元素
- 拖拽平移画布
- 滚轮缩放画布（0.25x — 2x）
- 双击空白处新建项目
- 右键上下文菜单（新建、粘贴、视图操作）
- 点阵网格背景

### 1.2 项目管理
- 创建新项目（自动分配颜色和名称）
- 重命名项目
- 复制项目
- 归档/删除项目（运行时确认）
- 项目间依赖连线（可视化箭头）
- 项目列表侧边栏（可折叠）

### 1.3 画布视图控制
- 全局命令面板（⌘+K）
- 快速定位到指定项目
- 画布缩略图导航（概览模式）
- 全屏演示模式

---

## 2. 项目卡片

### 2.1 卡片基础功能
- 展开/折叠项目（保持运行状态）
- 拖拽自由摆放
- 显示项目名称、状态指示灯、整体进度
- 右上角控制：最小化（到画布最小化栏）、最大化、关闭
- 卡片缩放时简化显示（过小显示为色块）

### 2.2 项目进度聚合
- 聚合显示所有子 Terminal 的任务进度
- 显示活跃 Agent 数量
- 阻塞状态高亮（红色）
- 等待决策状态高亮（橙色）

---

## 3. 终端 (Terminal)

### 3.1 终端基础功能
- 每个 Terminal 严格属于一个 Agent
- 支持多 Terminal 并行运行（同一项目内）
- Terminal 重排顺序
- 关闭Terminal（确认）
- 清空历史
- 导出日志

### 3.2 终端交互
- 消息输入（支持多行）
- 终端输出显示（时间戳 + 类型图标）
- 搜索终端历史（⌘+F）
- 自动滚动跟随输出
- 手动滚动时暂停自动跟随
- 拆分视图（左右/上下）

### 3.3 终端菜单
- 清空
- 重放
- 导出日志
- 拆分视图
- 查看 Agent 信息

---

## 4. 多 Agent 系统

### 4.1 Agent 管理
- 每个 Terminal 绑定一个 Agent 实例
- Agent 类型：planner、coder、reviewer、tester 等
- Agent 配色方案（唯一标识色）
- Agent 状态：initialized、thinking、running、idle、waiting_decision、completed、error

### 4.2 多 Agent 协作
- 同一项目内多个 Agent 分工
- Agent 之间共享项目状态
- 父 Agent 可分配子任务给其他 Agent
- 子 Agent 进度实时同步到父 Agent
- Agent 通信日志可见

### 4.3 Agent 可视化
- Agent 头像（几何图形 + 配色）
- Agent 状态动画（thinking 脉冲、running 旋转）
- 子 Agent 任务列表在父 Agent Terminal 内展开

---

## 5. 任务进度系统

### 5.1 任务状态
```
Pending → In Progress → Completed
              ↓
       Waiting Decision → Approved / Rejected
              ↓
           Blocked (rejected)
```

### 5.2 任务展示
- 项目内任务列表
- 每个任务显示：状态图标 + 描述 + 负责人 Agent
- 进度百分比
- 阻塞原因提示

### 5.3 进度计算
- 项目整体进度自动聚合
- 支持任务权重配置
- 阻塞任务单独高亮

---

## 6. 决策系统 (Human-in-the-Loop)

### 6.1 决策触发
- Agent 在关键操作前主动暂停并请求确认
- 决策点类型：文件修改、执行危险命令、多文件改动、删除操作

### 6.2 决策显示
- 决策请求显示在悬浮窗（Mini Widget）中，不打断当前工作流
- 显示：项目名、Agent、Terminal、决策内容摘要
- Diff 预览（当前状态 vs 提议变更）
- 决策选项：批准 / 拒绝 / 编辑后批准
- 悬浮窗中支持快捷键（⌘+Enter 批准，Esc 拒绝）
- 决策卡片置顶显示，始终可见

### 6.3 决策历史
- 所有决策记录可追溯
- 支持查看决策内容和结果
- 支持撤销决策（有限窗口）

---

## 7. 通知系统

### 7.1 悬浮小窗 (Mini Widget)
- 最小化时显示在屏幕角落（可拖拽）
- 滚动显示所有项目进度
- 每个项目显示：名称、进度条、百分比、活跃 Agent 数量
- 决策卡片置顶显示

### 7.2 通知类型
- 子 Agent 完成任务通知
- 需要人类决策弹出提醒
- 项目出错/失败警示
- 长时间无活动静默

### 7.3 通知交互
- 点击项目名快速定位到画布
- 点击决策卡片展开详情
- 任务栏/系统通知集成（可选）

---

## 8. 状态与持久化

### 8.1 会话状态
- 画布布局保存与恢复
- 正在运行的任务状态保留
- 终端历史保留
- 项目进度保留

### 8.2 持久化（后续扩展）
- 项目配置持久化
- Agent 配置持久化
- 用户偏好持久化

---

## 9. 快捷键系统

| 快捷键 | 功能 |
|--------|------|
| ⌘+N | 新建项目 |
| ⌘+W | 关闭当前 Terminal |
| ⌘+Tab | 切换 Terminal |
| ⌘+Shift+\\ | 新建 Terminal（当前项目） |
| ⌘+Enter | 批准当前决策 |
| Esc | 拒绝当前决策 / 关闭弹窗 |
| ⌘+K | 全局命令面板 |
| ⌘+F | 搜索（上下文相关） |
| ⌘+B | 切换侧边栏 |
| ⌘+M | 最小化到悬浮窗 |

---

## 10. 性能目标

- 支持 100+ Terminal 同时运行
- 画布缩放/拖拽保持 60fps
- 终端输出滚动无卡顿
- 决策弹窗响应 < 100ms
