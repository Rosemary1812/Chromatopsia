# @chromatopsia/canvas

Chromatopsia Canvas — 无限画布基础设施，包含画布渲染、悬浮球、侧边栏、语音输入。

## 模块

| 模块 | 说明 |
|------|------|
| `canvas/` | 无限画布渲染引擎 |
| `floating/` | 悬浮球 Mini Widget |
| `sidebar/` | 侧边栏（文件树 / Diff 视图 / MD 渲染） |
| `voice/` | 语音输入（ASR Adapter + Command Parser） |
| `components/` | 共享 UI 组件 |

详细设计文档见 `../../../Program/architecture/README.md`。
