# @chromatopsia/ui-shell

Chromatopsia UI Shell — 画布、悬浮窗、侧边栏等外围 UI 组件。

## 模块

| 模块 | 说明 |
|------|------|
| `canvas/` | 无限画布渲染 |
| `floating/` | 悬浮窗 Mini Widget |
| `sidebar/` | 侧边栏（文件树 / Diff / MD 渲染） |
| `voice/` | 语音输入 |
| `components/` | 共享 UI 组件 |

详细设计文档见 `../../../Program/architecture/README.md`。
